## Overview [Copied!](#overview "Copy link to this section")

### AppSec dashboards [Copied!](#appsec-dashboards "Copy link to this section")

The Cycode platform offers AppSec dashboards designed around the core pillars of Application Security Posture Management: **Visibility**, **Prioritization**, and **Remediation**. These dashboards provide quick overviews and actionable insights, offering full visibility into your organization’s security landscape. They help you efficiently triage and prioritize issues, while providing a clear understanding of remediation progress and overall security posture.

*   [Visibility dashboard](/dashboards/cycode-360/visibility-dashboard)
    
*   [Prioritization dashboard](/dashboards/cycode-360/prioritization-dashboard)
    
*   [Remediation dashboard](/dashboards/cycode-360/remediation-dashboard)
    

### Scanner dashboards [Copied!](#scanner-dashboards "Copy link to this section")

Cycode also provides dedicated dashboards for Secrets, SCA, and SAST scanners. These focused dashboards deliver insights specific to each category, empowering teams to address risks and optimize security efforts in these critical areas.

*   [Secrets dashboard](/dashboards/scanner-dashboards/secrets-dashboard)
    
*   [SCA dashboard](/dashboards/scanner-dashboards/sca-dashboard)
    
*   [SAST dashboard](/dashboards/scanner-dashboards/sast-dashboard)
    

### Threat intelligence [Copied!](#threat-intelligence "Copy link to this section")

The Threat Intelligence dashboard features the must current threats reported by Cycode’s research team, so you can stay up to date on the latest industry threats and zero-day exploits.

## Filter by project [Copied!](#filter-by-project "Copy link to this section")

If you’re assigned to more than one project, you can filter each dashboard according to your available projects. If you’re only assigned to a single project, this filter isn’t shown.
