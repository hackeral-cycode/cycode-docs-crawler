# Sumo Logic

Integrating Sumo Logic enables Cycode to send information about detected violations in your environments.

After successfully integrating, you can organize and track relevant information in your monitoring service.

## Permissions [Copied!](#permissions "Copy link to this section")

You’ll need the following permissions to integrate Sumo Logic with Cycode.

Permissions

Why is write access required?

Read/write permissions

To create a ticket or alert.

Admin permissions

Cycode only integrates with Sumo Logic accounts that have administrator permissions.

## Connect Sumo Logic to Cycode [Copied!](#connect-sumo-logic-to-cycode "Copy link to this section")

1.  Go to **Settings > Integrations** and then click **Add Integration**.
    
2.  In the **Alerting** section, click **Sumo Logic**.
    
3.  In the connector wizard, type a name for your integration.
    
4.  Configure an HTTP source address. See [Sumo](https://help.sumologic.com/03Send-Data/Sources/02Sources-for-Hosted-Collectors/HTTP-Source/zGenerate-a-new-URL-for-an-HTTP-Source) for additional details.
    
5.  Enter your source address URL in Cycode.
    
6.  Select the Cycode policies you want to receive alerts about in Sumo Logic.
    
7.  Click **Done**. After you click **Done**, the Sumo Logic integrations page shows that your integration is connected.
    

To add another integration, click **Add Integration** in the Sumo Logic integrations page.
