## Overview [Copied!](#overview "Copy link to this section")

This topic explains how to integrate a Slack channel so that Cycode can send information about violations that are detected in your environment.

## Prerequisites [Copied!](#prerequisites "Copy link to this section")

### General [Copied!](#general "Copy link to this section")

Make sure you have an available Slack channel where Cycode can post notifications.

Additionally, if the Cycode app isn’t installed, you’ll be prompted to install it. This action requires admin approval.

## Integrate Slack with Cycode [Copied!](#integrate-slack-with-cycode "Copy link to this section")

1.  Go to **Settings > Integrations** and click **Add Integration**.
    
2.  In the **Alerting** section, click **Slack**.
    
3.  In the **Use case** field, describe why you are connecting to this Slack channel and click **Add**.
    
4.  When you are redirected to Slack, sign in to the Slack workspace you are integrating. ![](/assets/slack-sign-in.png)
    
5.  Find and select the channel where Cycode should send notifications, and click **Allow**. When the integration is complete, this channel is displayed in the list of connected channels.
