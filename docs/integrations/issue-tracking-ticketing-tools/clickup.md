## Overview [Copied!](#overview "Copy link to this section")

Integrating with ClickUp enables Cycode to send information about detected violations in your environments. After successfully integrating, you can organize and track relevant information in your monitoring service.

Note

This is a one-way integration from the third-party API to Cycode.

### Prerequisites [Copied!](#prerequisites "Copy link to this section")

The ClickUp integration requires a Personal API Token (PAT). For more information about the PAT, go to the [ClickUp documentation](https://clickup.com/api/developer-portal/authentication#personal-token) .

## Integrate with ClickUp [Copied!](#integrate-with-clickup "Copy link to this section")

1.  Go to **Settings > Integrations** and click **Add Integration**.
    
2.  In the **Issue Tracking** section, click **ClickUp**.
    
    ![](/assets/b70d0d5-image.png)
    
3.  In the connection wizard, fill in the following fields:
    
    1.  Integration Name - The name of the integration in Cycode.
    2.  Personal API Token (PAT) - The ClickUp personal API token.
    3.  Team ID - The Team ID for the ClickUp account.
    
    ![](/assets/058b17b-clickup-details.png)
    
4.  After you fill in all the relevant information, click **Done**. Cycode validates the credentials and information provided are working before completing the setup.
