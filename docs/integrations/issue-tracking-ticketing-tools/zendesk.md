## Overview [Copied!](#overview "Copy link to this section")

Integrating with Zendesk enables Cycode to send information about detected violations in your environments. After integration, you can organize and track relevant information in your monitoring service.

Note

This is a one-way integration from the third-party API to Cycode.

## Integrate with Zendesk [Copied!](#integrate-with-zendesk "Copy link to this section")

1.  Go to **Settings > Integrations** and click **Add Integration**.
    
2.  In the **Issue Tracking** section, click **Zendesk**.
    
    ![](/assets/b70d0d5-image.png)
    
    ![](/assets/new-zendesk.png)
    
3.  Complete the relevant fields.
    
    Note
    
    Cycode recommends using an API token from your Zendesk account over a credentials set.
    
4.  Click **Done**.
