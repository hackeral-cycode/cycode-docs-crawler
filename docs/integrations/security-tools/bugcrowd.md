## Overview [Copied!](#overview "Copy link to this section")

This connector syncs bug bounty findings from Bugcrowd to Cycode.

Note

This is a one-way integration from the third-party API to Cycode.

## Domain [Copied!](#domain "Copy link to this section")

Bug bounty

## Connect Bugcrowd to Cycode [Copied!](#connect-bugcrowd-to-cycode "Copy link to this section")

1.  Provide a name for the integration.
2.  Get the following parameters from Bugcrowd and apply them in the Cycode connector wizard.
    
    Parameter
    
    Value
    
    **API username**
    
    Provide the username of the Bugcrowd API credentials. To create API credentials for the integration, follow the instructions in the [Bugcrowd docs](https://docs.bugcrowd.com/api/getting-started/) .
    
    **API password**
    
    Provide the password of the Bugcrowd API credentials.
    
3.  Click **Connect**.
