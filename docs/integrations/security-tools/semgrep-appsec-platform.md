## Overview [Copied!](#overview "Copy link to this section")

Cycode’s connector with the Semgrep Appsec Platform enables you to pull SAST vulnerabilities from Semgrep and correlate them in Cycode.

Note

This is a one-way integration from the third-party API to Cycode.

## Domain [Copied!](#domain "Copy link to this section")

SAST

## Permissions [Copied!](#permissions "Copy link to this section")

Role

Description

Web API

The API supports authentication with an API token with the “Web API” permission, without limited scopes of access.

## Setup [Copied!](#setup "Copy link to this section")

1.  Sign in to your Semgrep Appsec Platform account.
2.  Go to **Settings** > **Token** > **API tokens** > **Create new token**.  
    ![](/assets/image%28634%29.png)
3.  Set the scope for **Web API** and provide indicative name for the token.  
    ![](/assets/image%28635%29.png)
4.  Store the token somehwere safe and click **Save**.
