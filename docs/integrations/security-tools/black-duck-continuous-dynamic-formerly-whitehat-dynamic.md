## Overview [Copied!](#overview "Copy link to this section")

This connector syncs DAST vulnerabilities from Black Duck Continuous Dynamic to Cycode.

Note

This is a one-way integration from the third-party API to Cycode.

## Domain [Copied!](#domain "Copy link to this section")

DAST

## Connect Black Duck Continuous Dynamic to Cycode [Copied!](#connect-black-duck-continuous-dynamic-to-cycode "Copy link to this section")

Get the following parameters from Black Duck Continuous Dynamic and apply them in the Cycode connector wizard.

Parameter

Value

API key

The Black Duck Continuous Dynamic API Key.  
  
To create or get your API key, in Black Duck go to **Continuous Dynamic Portal** > **My Profile** > **API Key**.

## Permissions [Copied!](#permissions "Copy link to this section")

Requirement

Description

Valid API key

No special permissions are required, because each API key has access to all vulnerability data.
