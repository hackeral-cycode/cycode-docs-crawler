## Overview [Copied!](#overview "Copy link to this section")

Cycode’s connector for Fortify SSC enables pulling vulnerabilities discovered by Fortify so you can view and manage them in Cycode.

Note

This is a one-way integration from the third-party API to Cycode.

## Domain [Copied!](#domain "Copy link to this section")

SAST

## Connect Fortify SSC to Cycode [Copied!](#connect-fortify-ssc-to-cycode "Copy link to this section")

Configure the following parameters in the Cycode connector wizard, then click **Connect**.

Parameter

Value

Server URL

The Fortify SSC server URL.

API token

A Fortify SSC API token, generated from **Administration** > **Users** > **Token Management**.  
  
**Note**: The new token should be of type **AutomationToken**.

Broker

(Optional) If your environment uses Cycode Broker, select the Broker for this integration.

Violation types

Select the type(s) of scanning for this integration.
