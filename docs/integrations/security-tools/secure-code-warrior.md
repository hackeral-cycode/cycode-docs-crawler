Integration Limitations

This integration works by mapping the CWE and detected coding language between Cycode violations and Secure Code Warrior. Violations that don’t include a CWE or use a language not supported by Secure Code Warrior will not work with this integration.

## Overview [Copied!](#overview "Copy link to this section")

The Secure Code Warrior integration allows you to enrich Cycode’s SAST remediation guidelines with videos and interactive exercises. This content can be applied to the Cycode violation card, in the developer’s IDE, or to the Cycode comment in the relevant PR.

## Domain [Copied!](#domain "Copy link to this section")

API Security

## Prerequisites [Copied!](#prerequisites "Copy link to this section")

To enable Secure Code Warrior exercise content, a valid Secure Code Warrior license is required. Users will be redirected to the Secure Code Warrior platform and prompted to log in.

Note

You don’t need a Secure Code Warrior license to access video content. However, a license is required for full access to their interactive exercise platform.

## FAQs [Copied!](#faqs "Copy link to this section")

### Do I need my Secure Code Warrior API key for any part of this integration? [Copied!](#do-i-need-my-secure-code-warrior-api-key-for-any-part-of-this-integration "Copy link to this section")

No, you don’t need to use your API key. Cycode can automatically display video content and provide links to exercises from the Secure Code Warrior platform. If you have a Secure Code Warrior license, your users can sign in using their regular Secure Code Warrior credentials to access the exercises.
