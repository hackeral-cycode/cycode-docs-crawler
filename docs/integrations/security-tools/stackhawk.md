## Overview [Copied!](#overview "Copy link to this section")

This connector syncs DAST vulnerabilities from StackHawk to Cycode.

Note

This is a one-way integration from the third-party API to Cycode.

## Domain [Copied!](#domain "Copy link to this section")

DAST

## Connect StackHawk to Cycode [Copied!](#connect-stackhawk-to-cycode "Copy link to this section")

1.  Provide a name for the integration.
    
2.  Get the following parameters from StackHawk and apply them in the Cycode connector wizard.
    
    Parameter
    
    Value
    
    **API key**
    
    To get the API key:In the StackHawk interface, go to [**Settings** > **API Keys**](https://app.stackhawk.com/settings/apikeys) .Click **Create API Key**.
    
    **Organization ID**
    
    To get the Organization ID:In the StackHawk interface, go to [**Settings** > **Organization Details**](https://app.stackhawk.com/settings/organizationdetails) .Copy the **ORG ID**.
    
3.  In the Cycode connector wizard, click **Connect**.
