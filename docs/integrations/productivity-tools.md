## Overview [Copied!](#overview "Copy link to this section")

Cycode can detect and alert on exposed secrets in blogs, issues, comments, pages, and files stored across your productivity tools to help organizations remediate risks before they can be exploited.

## Supported productivity tools [Copied!](#supported-productivity-tools "Copy link to this section")

Cycode currently integrates with with the following productivity tools:

*   [Confluence Cloud](/integrations/productivity-tools/confluence-cloud)
    
*   [Confluence Data Center](/integrations/productivity-tools/confluence-data-center)
    
*   [Jira Cloud](/integrations/productivity-tools/jira-cloud)
    
*   [Jira Data Center](/integrations/productivity-tools/jira-data-center)
    
*   [Microsoft SharePoint Online](/integrations/productivity-tools/microsoft-sharepoint-online)
    
*   [Microsoft Teams](/integrations/alerting-tools/microsoft-teams)
    
*   [Slack](/integrations/productivity-tools/slack)
