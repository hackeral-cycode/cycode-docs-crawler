## Overview [Copied!](#overview "Copy link to this section")

When you connect a third-party security tool to the Cycode platform, Cycode retrieves the data from the tool regarding security vulnerabilities and displays it in the Cycode dashboards. You can then analyze and manage these violations inside Cycode, providing easy access to all your security issues in a single place, and enabling a more comprehensive view of the overall security posture in your organization.

### Available connectors [Copied!](#available-connectors "Copy link to this section")

You can create a custom connector to integrate Cycode with a custom in-house security tool. For more information, go to [Create a custom connector](/integrations/security-tools/create-a-custom-connector/) .

You can also connect the Cycode platform to any of the following third-party security tools.
