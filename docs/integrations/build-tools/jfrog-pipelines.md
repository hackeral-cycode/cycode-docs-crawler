## Overview [Copied!](#overview "Copy link to this section")

JFrog Pipeline is an automated solution for building, testing, and deploying software as part of the CI/CD pipeline. The integration of JFrog Pipelines with Cycode enables scanning code, artifacts, and the CI/CD pipeline for secrets, misconfigurations, and leaks thanks to [Cycode’s Context Intelligence Graph](/context-intelligence-graph/) .

The Context Intelligence Graph aggregates asset information, events, metadata, and user activity from DevOps tools, infrastructure, and security scanners across your entire SDLC. It then correlates seemingly disparate events to create contextual insights, prioritize remediation, reduce false positives, and ensure the integrity of the pipeline and help to mitigate the risk of attacks through code tampering.

## Permissions [Copied!](#permissions "Copy link to this section")

Pipelines are part of the JFrog integration, and use the same permissions as the [JFrog Artifactory and Xray](/integrations/security-tools/jfrog-artifactory-and-xray/#permissions) integrations.

## Integrate JFrog Pipelines with Cycode [Copied!](#integrate-jfrog-pipelines-with-cycode "Copy link to this section")

To integrate JFrog Pipelines with Cycode, follow the steps described in [JFrog Artifactory and Xray](/integrations/security-tools/jfrog-artifactory-and-xray/#connect-jfrog-to-cycode) .
