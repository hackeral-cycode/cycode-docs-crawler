# Admin settings

[](https://gitlab.com/cycodehq/documentation/cycode_docs/-/blob/main/src/content/cycode-settings/admin-settings/_index.md "Edit this page")

Updated on Apr 24, 2025 • Published on Jul 29, 2024

* * *

If have the Admin or Owner role assigned in Cycode, you can access the admin settings in the Settings view.

Note

The Settings link has been moved from the Profile dropdown area to the main Cycode navigation pane.

You can configure the following organization-level settings:

*   [AI features](/cycode-settings/admin-settings/ai-features) in the platform
    
*   [Plan details](/inventory/code-entities/organizations)
    
*   [Project settings](/cycode-settings/admin-settings/projects)
