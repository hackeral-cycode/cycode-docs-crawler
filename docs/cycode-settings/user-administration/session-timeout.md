# Session timeout

[](https://gitlab.com/cycodehq/documentation/cycode_docs/-/blob/main/src/content/cycode-settings/user-administration/session-timeout.md "Edit this page")

Updated on Mar 23, 2025 • Published on Jul 7, 2024

* * *

Administrators can configure a session timeout to ensure that members don’t stay signed in to Cycode if they aren’t actually using the system.

**To configure the session timeout value**

1.  Go to **Settings > Session Time-out**.
    
2.  To enable session timeout, toggle on the option.
    
3.  To set the timeout value, select one of the preset values. The default is **5 min**.
