# Single Sign-on (SSO)

[](https://gitlab.com/cycodehq/documentation/cycode_docs/-/blob/main/src/content/cycode-settings/user-administration/single-sign-on-sso/_index.md "Edit this page")

Updated on Mar 23, 2025 • Published on Jun 24, 2024

* * *

Cycode provides Single Sign On (SSO) capability over SAML. Role mapping functionality and session timeout configuration are available starting with Cycode user management v2, and the following is also supported:

*   User data
    
*   Social login
    
*   SSO Identity Provider configuration
    
*   Passing the Tenant ID in JWT claims (via Groups)
    

Cycode manages the following in its AuthService:

*   Current tenant
    
*   User roles
    

This section provides instructions on how to configure SSO, connect your SSO over SAML, and set up role mapping.
