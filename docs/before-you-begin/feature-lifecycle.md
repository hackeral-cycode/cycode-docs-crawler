# Feature lifecycle

[](https://gitlab.com/cycodehq/documentation/cycode_docs/-/blob/main/src/content/before-you-begin/feature-lifecycle.md "Edit this page")

Updated on Apr 26, 2026 • Published on Apr 26, 2026

Cycode continuously ships new capabilities across the platform. To ensure new functionality is stable and meets your expectations before wide release, significant features go through a structured maturity process before becoming Generally Available (GA).

## Release Stages [Copied!](#release-stages "Copy link to this section")

### Private Access [Copied!](#private-access "Copy link to this section")

Private Access features are enabled for a select group of customers by the Cycode team. Access is by invitation only and is not self-serve.

*   **Expectations:** The feature is functional but actively being refined. Your feedback directly impacts the feature’s development.
*   **Access:** Contact your Cycode account team to request participation.

### Early Access [Copied!](#early-access "Copy link to this section")

Early Access features are available to all customers via an opt-in toggle. These features have been validated through internal testing or Private Access and are stable enough for broader use.

*   **Expectations:** The feature is opt-in and not enabled by default. Documentation is available and marked with an **Early Access** badge.
*   **Access:** Requires Admin permissions to enable via the **Settings** menu.

### Generally Available (GA) [Copied!](#generally-available-ga "Copy link to this section")

GA features are fully released and enabled for all eligible customers. They meet Cycode stability and quality standards and are backed by full support.

*   **Expectations:** The feature is enabled by default. Standard Cycode SLAs and support apply.
*   **Access:** Automatically available to all users.

* * *

## How to Enable Early Access Features [Copied!](#how-to-enable-early-access-features "Copy link to this section")

Admins can manage the availability of pre-GA features within the platform.

1.  Navigate to **Settings** > **Admin Settings**.
2.  Locate the **Early Access** section.
3.  **Toggle** the switch to **On**.
4.  **Click** **Save**.

Note

Disabling the **Early Access** setting may hide associated features data or UI elements. No data is deleted, but the functionality will be restricted until re-enabled.
