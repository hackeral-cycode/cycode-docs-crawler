# How do I know if my Connector X integration is working properly?

[](https://gitlab.com/cycodehq/documentation/cycode_docs/-/blob/main/src/content/knowledge-base/helpful-tips-and-best-practices/how-do-i-know-if-my-integrations-are-working-properly/how-do-i-know-if-my-connector-x-integration-is-working-properly.md "Edit this page")

Updated on Jul 14, 2024 • Published on Jul 7, 2024

* * *

Once you’ve completed an integration using Connector X, it may take time for the data to be collected and for all Cycode actions to be completed (there’s currently no indication that the integration is complete).

The results will be viewable first in the Risk Intelligence Graph (RIG) and only after that in the Violations page. Therefore, we recommend taking a few minutes until you see the violations from the Connector in the Violations page, which will be a final indication that the integration process is complete.
