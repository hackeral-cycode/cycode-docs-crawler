## What’s the difference between the “Detected At” vs. “Issue creation date” vs. “Introduced At” timestamps? [Copied!](#whats-the-difference-between-the-detected-at-vs-issue-creation-date-vs-introduced-at-timestamps "Copy link to this section")

## Answer [Copied!](#answer "Copy link to this section")

**Detected at** indicates when Cycode detected the violation.  
**Introduced at** means when the vulnerability was created (in the code for example)

For example: when running an intial scan and finding a secret, the ‘detected at’ will be according to this time (more or less), but the secret could have been first been introduced in the repository a year ago.
