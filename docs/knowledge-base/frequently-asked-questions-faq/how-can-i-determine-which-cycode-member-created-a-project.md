# How can I determine which Cycode member created a project?

[](https://gitlab.com/cycodehq/documentation/cycode_docs/-/blob/main/src/content/knowledge-base/frequently-asked-questions-faq/how-can-i-determine-which-cycode-member-created-a-project.md "Edit this page")

* * *

## Answer [Copied!](#answer "Copy link to this section")

Go to the Cycode Audit Logs and apply the following filters:

*   Include Cycode automation=No
*   Resource Type=Cycode Project
*   Action=Created

The result will display the Cycode member who created the Cycode project and the date it was created.
