# Common actions in Cycode

[](https://gitlab.com/cycodehq/documentation/cycode_docs/-/blob/main/src/content/knowledge-base/frequently-asked-questions-faq/common-actions-in-cycode.md "Edit this page")

* * *

When navigating Cycode, some actions appear in multiple places. These include:

*   **Search** - search sections of Cycode for specific things such as policy names, organizations, and repositories
*   **Policy Settings** - navigate to the Policies page, filtered according to the currently displayed violations.
*   **Export Data** - export the current data from a section of Cycode as a CSV file
*   **Resolve** - resolve a policy violation alert after the violation is fixed
*   **Dismiss** - dismiss a policy violation alert when you don’t want to see the alert anymore
*   **Reopen**\- reopen a policy violation alert. The alert status changes to **Open**
*   **Create Ticket** - create a ticket in an integrated tracking platform to organize and track the relevant policy violation
