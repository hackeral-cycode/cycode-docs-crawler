**Q:** Does the aggregation option can be applied in a workflow for MS Teams integration?  
**A:** No, there is currently a known limitation with the Aggregation feature for Workflows where it can only be applied to the following Alerting types:

*   Email
*   Slack
*   Jira (Cloud)
*   Jira Server (Data Center)
