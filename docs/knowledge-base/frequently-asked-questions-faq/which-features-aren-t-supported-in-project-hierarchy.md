## Answer [Copied!](#answer "Copy link to this section")

The project hierarchy feature is still in its early stages, and additional functionalities will be added soon, including:

*   Search by group name
*   More data in group tables (e.g., risk, total violations, MTTR)
*   Ability to select a parent group from the project creation wizard
*   Display of group entities in the graph
