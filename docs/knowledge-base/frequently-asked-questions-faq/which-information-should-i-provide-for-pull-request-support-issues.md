# Which information should I provide for pull request support issues?

## Answer [Copied!](#answer "Copy link to this section")

When reporting issues with Pull Requests, please be sure to provide the following info:

*   The type of Status Check scan being used (i.e., Secret, IaC, etc.)
*   The repository of the affect pull request.
*   The Pull Request number
*   Optionally, the link to the pull request, as it contains the repository and pull request number

These details will allow us to search from our end and determine if there is an issue with the specific scan or a system-wide issue.
