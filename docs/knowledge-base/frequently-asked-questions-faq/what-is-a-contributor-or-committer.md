# What is a contributor or committer?

[](https://gitlab.com/cycodehq/documentation/cycode_docs/-/blob/main/src/content/knowledge-base/frequently-asked-questions-faq/what-is-a-contributor-or-committer.md "Edit this page")

* * *

## Answer [Copied!](#answer "Copy link to this section")

A contributor or committer is defined as someone who’s permitted to modify the source code of a software project by committing the code. Therefore, when generating a RIG query, you’ll use SCM active committers to get the number of contributors.
