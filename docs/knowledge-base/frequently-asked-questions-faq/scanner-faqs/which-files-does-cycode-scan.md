Warning

Both lists are not exhaustive and only contain a few example file types as reference.

When integrating an SCM tool with Cycode, we will scan all the content that appears there with a textual format. For example, the following list is some common file types we scan:

*   \*.json
*   \*.ts
*   \*.txt
*   \*.java
*   \*.py
*   \*.xml
*   \*.js

Cycode does not scan any file types that do not fall under a textual format, which includes the following list:

*   \*.pdf
*   \*.bin
*   \*.exe
*   \*.zip
