## Overview tab [Copied!](#overview-tab "Copy link to this section")

The Overview tab contains the following sections.

Section

Description

Summary info

A collection of tiles that highlight important information for that type of violation.

Code snippet

(Not shown for all violations) When available, this area displays the code snippet where the violation was detected.

Description

(Not shown for all violations) A general description of why this finding is a violation.

General info

The general information displayed here is specific to the type of violation detected.

Exposure path

(Not shown for all violations) The exposure path shows where the violation is located, where it is potentially exposed, and the relationship between the points.

Labels

All of the labels associated with this violation are shown. You can also add a custom label.

Additional info

The additional information displayed here is specific to the type of violation detected.
