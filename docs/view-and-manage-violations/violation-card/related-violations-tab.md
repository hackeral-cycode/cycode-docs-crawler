## Overview [Copied!](#overview "Copy link to this section")

The Related violations tab lists the detected violations in Cycode that share traits or root causes, such as having the same secret value or existing in the same repository.

### Violation details [Copied!](#violation-details "Copy link to this section")

The following information is displayed in the table that lists the related violations.

Column

Description

Visible by default

Violation status

Current status of the related violation.

Yes

Risk score

Custom score assigned by Cycode based on specific risk factors.

Yes

Policy and category

Name and category of the policy that was used to detect the violation.

Yes

Related reason

Reason this violation is related to the original violation.

No

Found in

Platform or SCM where the related violation was detected.

Yes

Author name

Username of the organization member that committed the code that contains the related violation.

No

Last detected

Date and time of the last scan that detected the related violation.

No

Status change message

If the status of the related violation changed when it was last detected, this column contains additional information about the status change.

No
