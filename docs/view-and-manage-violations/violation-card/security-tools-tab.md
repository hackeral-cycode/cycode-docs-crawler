## Overview [Copied!](#overview "Copy link to this section")

Cycode supports the integration of additional security tools, which enables us to also show violations detected by those other tools. If you have any third-party security tools integrated with Cycode, you’ll also see an additional Security Tool tab at the top, which shows you the resulting violations from the integrated tool.

## Tab details [Copied!](#tab-details "Copy link to this section")

The Security tab shows the vulnerability findings according to security tool and package. If there are multiple entries, expand an entry to display details about the vulnerability and the vulnerable asset. The information displayed depends on the security tool.
