## Comments tab [Copied!](#comments-tab "Copy link to this section")

This interactive tab contains comments on the detected violation; any user with access to this page can leave a comment.

![](/assets/image-1724658898161.png)

Note that you can comment on any asset type—not just violations, but also repositories, organizations, and users.

To add a new comment, enter the comment in the “Add a comment” field (highlighted above) and click the arrow.
