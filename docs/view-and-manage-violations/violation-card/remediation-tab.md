## Remediation tab [Copied!](#remediation-tab "Copy link to this section")

Many Cycode policies (but not all) include the Cycode default remediation guidelines.

*   If a policy has only Cycode remediations, then only those will be shown here.
*   If a policy has only company-defined remediation guidelines, then only those will be shown in the card.
*   If a policy has both a Cycode remediation and a company-defined one, then you’ll be able to see both options in two tabs, as shown in the image below.

If there are no remediations of any type, you can also use this tab to add them to the violation for future use by clicking **Add Guidelines**

![](/assets/Remediation-Doc-image.png)

Note

Allowed user roles for updating company guidelines: Admin, Owner

If there are no remeditions available for the violation, the Remediation tab will appear greyed-out; standing on the tab will display a notice “No remediation guidelines available” as well as a link to add company remediation guidelines (which will open the tab itself).

![](/assets/image%28219%29.png)

### Gathering and storing feedback about remediations [Copied!](#gathering-and-storing-feedback-about-remediations "Copy link to this section")

Displayed at the end of every Cycode remediation, users can provide feedback about remediations by selecting either the thumbs up or thumbs down button.

### Editing (or deleting) company remediation guidelines [Copied!](#editing-or-deleting-company-remediation-guidelines "Copy link to this section")

You can edit company remediation guidelines (or delete them) using the three-dot action menu to the right of the guidelines.
