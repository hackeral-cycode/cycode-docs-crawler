## Overview [Copied!](#overview "Copy link to this section")

Note

This feature is currently in early access phase.

You can view dashboards from your collection-custom-dashboards in the dashboard workspace.

## View collections [Copied!](#view-collections "Copy link to this section")

1.  Go to **Dashboards** > **My dashboards** > **Dashboard workspace**.
    
2.  Click a collection to the left of the table to view the contents. The following information is displayed.
    
    Column
    
    Description
    
    **Type**
    
    Type of object in the collection, either a question (widget) or a dashboard.
    
    **Name**
    
    Name of the object.
    
    **Last edited by**
    
    User that applied the most recent changes to the object.
    
    **Last edited at**
    
    Date and time that the most recent changes were applied.
    

## View a dashboard [Copied!](#view-a-dashboard "Copy link to this section")

You can select a dashboard from the public or private collection in the dashboard workspace, or you can select from the dashboards shown in the secondary navigation pane.

You can do the following when viewing a dashboard:

*   Use the dashboard and widget filters to streamline the data being displayed
    
*   Click into widgets (where available) for deeper analysis
    
*   Export the dashboard as a PDF
    

If you are an administrator, you can also edit a global dashboard.
