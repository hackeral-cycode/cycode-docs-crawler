## Overview [Copied!](#overview "Copy link to this section")

The scanner dashboards provide high-level information about scan results. You can filter the data using the global filters, and change the scoring method used to view the data between risk and severity.

## Available dashboards [Copied!](#available-dashboards "Copy link to this section")
