# Connect your first integration

[](https://gitlab.com/cycodehq/documentation/cycode_docs/-/blob/main/src/content/get-started/connect-your-first-integration.md "Edit this page")

Updated on Jan 25, 2026 • Published on Apr 23, 2025

* * *

You have to integrate with at least one source code management (SCM) tool to start measuring your security position. SCM tools provide a framework for developers and administrators to manage, track, organize and update changes to software code over time.

Choose one of the following SCM tools that fit your use case for the first integration. You can then add additional SCM integrations as per your organizational needs.

*   [GitHub](/integrations/scm-source-code-management-tools/github)
    
*   [GitHub Enterprise](/integrations/scm-source-code-management-tools/github-enterprise-server)
    
*   [GitLab](/integrations/scm-source-code-management-tools/gitlab)
    
*   [GitLab Self-Managed](/integrations/scm-source-code-management-tools/gitlab-self-managed)
    
*   [Bitbucket](/integrations/scm-source-code-management-tools/bitbucket)
    
*   [Bitbucket Server](/integrations/scm-source-code-management-tools/bitbucket-server)
    
*   [Azure DevOps](/integrations/scm-source-code-management-tools/azure-devops)
    
*   [Azure DevOps Server](/integrations/scm-source-code-management-tools/azure-devops-server)
    
*   [Gerrit](/integrations/scm-source-code-management-tools/gerrit)
    
*   [Helix Core](/integrations/scm-source-code-management-tools/perforce-p4-formerly-helix-core)
