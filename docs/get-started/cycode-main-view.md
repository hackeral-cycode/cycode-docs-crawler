## Overview [Copied!](#overview "Copy link to this section")

This topic describes the main Cycode view, with the various elements that help you navigate and refine the displayed data.

## Understand the Cycode view [Copied!](#understand-the-cycode-view "Copy link to this section")

The main view contains the following functionality to help you find and analyze the data that’s important to you.

![](/assets/cycode-view.png)

The following table describes the areas of the view that are numbered in the above image.

Number

Area

Description

1

Main tabs

Collection of available views. Includes the following:

*   [Dashboards](/dashboards)
*   [Violations](/view-and-manage-violations)
*   [Projects](/cycode-projects)
*   [Inventory](/inventory)
*   [Discovery](/context-intelligence-graph)
*   [Compliance](/compliance)
*   [Policies](/policies)
*   [Automation](/workflow-automation)
*   [Reports](/additional-cycode-features/generate-a-report)
*   [Developer activity](/developer-experience/developers)
*   [Settings](/cycode-settings)

The tabs that you see in your view depend on your access permissions in the platform.

2

Secondary navigation pane

Available for all main views that have nested views.

3

Tenant and project selector

Users that have access to multiple tenants can select the one they want to view. Additionally, users can select a single group or project from the project selector to view only data that is relevant for the selection. This affects all the tabs in the interface.

Note

The following tabs that display tenant-level data aren’t available when a project or group is selected here:

*   Discovery
    
*   Policies
    
*   Reports
    
*   Dev. activity
    

The Dashboard view is restricted to the Project views.

4

Workspace

Main area of the view that contains widgets, tables, and filters.

5

Menu bar

Provides access to the following:

*   [Notification center](/get-started/notification-center)
*   Product updates
*   Export queue
*   Contact us
*   Documentation
*   [Account settings](/get-started/account-and-personal-settings)
