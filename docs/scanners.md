# Scanners

Updated on Nov 11, 2025 • Published on Mar 26, 2024

* * *

Cycode offers a suite of advanced scanners to ensure comprehensive code security:

## Code

### SAST Scanner

Analyze source code to uncover and address security flaws early in the development process, before the code is ever deployed.

[Configure PR Scans →](/developer-experience/developers/pull-request-pr-scanning/sast-pr-scanner)

### SCA Scanner

Examine open-source components and their dependencies for known vulnerabilities to mitigate risks from third-party libraries.

[Configure PR Scans →](/developer-experience/developers/pull-request-pr-scanning/sca-pr-scanners)

### IaC Scanner

Review infrastructure configurations written in code for security issues and misconfigurations to ensure secure, compliant deployments.

[Configure PR Scans →](/developer-experience/developers/pull-request-pr-scanning/iac-pr-scanner)

### Secret Scanner

Identify sensitive information like passwords and API keys inadvertently embedded directly within your source code.

[Configure PR Scans →](/scanners/pipeline-security/secrets/configure-the-secret-scanner/configure-pr-scans)

## Build

### CI/CD Security

Secure the software delivery process by examining build configurations, deployment scripts, and automation workflows for risks.

[Configure PR Scans →](/developer-experience/developers/pull-request-pr-scanning/ci-cd-security-pr-scanner)

### CIMON

A specialized module for advanced pipeline security and monitoring, accessible via the platform's module settings.

[Configure PR Scans →](/scanners/pipeline-security/cimon/)

## Artifact

### Container Security

Scrutinize container images for vulnerabilities and misconfigurations to maintain integrity before they are released to production.

[Configure PR Scans →](/scanners/ast-application-security-testing/container-image-security/configure-the-container-security-scanner)

## Cloud

### Cloud Security

Evaluate cloud configurations, permissions, and deployments to mitigate risks and prevent unauthorized access to cloud resources.

[Configure PR Scans →](/scanners/)

### Leak Scanner

Detect potential data breaches by scanning repositories for exposed credentials, tokens, or private keys.

[Configure PR Scans →](/scanners/)
