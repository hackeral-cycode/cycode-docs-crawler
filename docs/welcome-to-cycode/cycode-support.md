At Cycode, we pride ourselves on providing excellent service to our customers in an effective and efficient manner. Our primary goal is to ensure 100% resolution of the issue and 100% satisfaction with the service provided.

## Submit a support request via email [Copied!](#submit-a-support-request-via-email "Copy link to this section")

You can submit a support request to Cycode support at [support@cycode.com.](mailto:support@cycode.com.)

## Open a support ticket from the Cycode platform [Copied!](#open-a-support-ticket-from-the-cycode-platform "Copy link to this section")

1.  To sign in to the Support Portal, click the **Contact Us** icon in the top right menu of the Cycode application.
2.  In the Support Portal, click **Submit new ticket**.
3.  Provide the details for the Support Ticket, as described [here](/welcome-to-cycode/cycode-support) .
    
    Item
    
    Description
    
    Tenant
    
    Specify the tenant if you have more than one.
    
    Issue
    
    Provide a brief summary of the issue, along with detailed steps to reproduce it (more is better). You can include relevant screenshots with context.  
    • For CLI related issues, always include the version and provide logs using the `--verbose` flag.  
    • For user-specific issues, include the username or email of the user.  
    • For PR issues, always include the PR link.  
    • For API-related issues, provide the call and response (omit your API Key before sending it to us).
    
    Project
    
    Specify the name of the affected project.
    
    Repository
    
    Specify the repository.
    
4.  Click **Submit**.
