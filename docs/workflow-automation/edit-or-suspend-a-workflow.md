## Overview [Copied!](#overview "Copy link to this section")

You can edit or suspend an existing workflow.

Important

You can’t delete a workflow.

## Edit an existing workflow [Copied!](#edit-an-existing-workflow "Copy link to this section")

1.  In the Automation view, find the workflow you want to update and select **Edit** from the More actions menu.
    
2.  Make your changes to the workflow.
    
3.  Test the changes to confirm that your changes have the required results, and then click **Publish**.
    

## Suspend a workflow [Copied!](#suspend-a-workflow "Copy link to this section")

When you suspend a workflow, Cycode no longer executes any of the automated tasks contained in the workflow.

Note

When you suspend a workflow, it only applies going forward. You can’t remove workflow actions retroactively for a suspended workflow.

*   In the Automation view, find the workflow you want to suspend and click the toggle to Disabled.
