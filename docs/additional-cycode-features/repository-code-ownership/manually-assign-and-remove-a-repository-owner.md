## Manually assigning and removing a repository owner [Copied!](#manually-assigning-and-removing-a-repository-owner "Copy link to this section")

In addition to Cycode’s assignment logic, you can also manually assign a repository owner in the Cycode platform.

In this case, you can assign owners per repository(s) - limited to 5 owners total.

*   In addition, you can edit ownership, and add/remove more owners.
    
*   When assigning users, you can search the select field according to the **SCM users** (only).
    
*   You can also remove an owner from the list and see all removed users and revert ownership.
    

1.  To manually assign ownership, go to **Inventory > Repositories**. ![](/assets/manually-assigning-and-removing-a-repository-owner-image-7fg2ff21.png)
    
2.  Open the relevant repository card:
    
3.  Click the edit icon next to the list of owners to display the Edit Repository owners dialog: ![](/assets/manually-assigning-and-removing-a-repository-owner-image-lyu4ej1z.png)
    
    The list of owners shows both the owners’ names and the method by which they were added (by Cycode or manually).
    
4.  To add another owner, in the search field, enter the name of the owner you wish to add (remember that you can only have up to 5 owners per repository).
    
5.  To remove an owner, click the delete button next to the owner’s name.
    
6.  To refresh the list, click the refresh button (this will not affect any owner’s you’ve already added).
