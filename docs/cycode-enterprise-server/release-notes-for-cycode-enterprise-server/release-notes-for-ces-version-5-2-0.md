### Secret scanning [Copied!](#secret-scanning "Copy link to this section")

#### New integrations [Copied!](#new-integrations "Copy link to this section")

Integrations are available for the following productivity tools, so that Cycode can scan them for secrets:

*   Microsoft Teams
*   Microsoft Sharepoint Online
*   Confluence Data Center
*   Jira Data Center

#### Generic secret verification using AI [Copied!](#generic-secret-verification-using-ai "Copy link to this section")

Cycode can verify detected generic secrets against our ML model to reduce false positives.  
In the **Settings** > **Admin settings** > **AI features** page, you can enable and disable the **Enable ML-based features** option.

Note

This feature is currently in beta. To enable it, please contact Support.

### Risk score [Copied!](#risk-score "Copy link to this section")

Cycode calculates the risk score for each detected violation according to a specific set of factors. The risk score varies depending on the values of these factors.

The risk score is displayed in the Violations view and the violations card.

Note

This feature is currently in beta. To enable it, please contact Support.

Limitation

The risk score feature is only available for new customers, because it can’t currently be added to existing customer tenants.
