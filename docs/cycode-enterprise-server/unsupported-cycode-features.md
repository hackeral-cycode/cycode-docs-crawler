# Unsupported Cycode features

## Overview [Copied!](#overview "Copy link to this section")

When Cycode develops and releases features, every effort is made to make them available for all deployment types. In some cases, some features may not be available for on-premise customers.

### General features [Copied!](#general-features "Copy link to this section")

*   AI features
*   Container scanning
*   Context Intelligence Graph

Automatic resolution of inactive secrets

If you are using Cycode on-premises (or in different service location than the one predefined in our system), the validation may not be accurate.

### Developer experience limitations [Copied!](#developer-experience-limitations "Copy link to this section")

*   SAST scanning in the VS Code Extension
*   SCA scanning is only partially supported in the VS Code Extension

### Unsupported integrations [Copied!](#unsupported-integrations "Copy link to this section")

*   Azure Pipelines - Self-hosted
*   Jenkins - Self-hosted - Kubernetes
*   ASPM Connectors:
    *   SonarQube (SAST)
    *   Semgrep Enterprise (SAST)
