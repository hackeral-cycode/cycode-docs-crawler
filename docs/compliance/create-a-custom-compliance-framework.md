## Overview [Copied!](#overview "Copy link to this section")

When you create a compliance framework, you have to define the following:

*   Requirements: The actions an organization must perform to comply with laws, regulations, or standards
    
*   Practices: Organizational actions and procedures implemented to meet requirements.
    
*   Controls: Specific measures, policies, and procedures that are put in place to ensure that the defined practices effectively meet the requirements.
    

## Create a custom framework [Copied!](#create-a-custom-framework "Copy link to this section")

When you create a framework, you have to add at least one practice, add at least one requirement to the practice, and then add at least one control to the requirement.

### Create the framework [Copied!](#create-the-framework "Copy link to this section")

1.  In the Compliance view, click **Custom Framework**.
    
2.  In the Framework Details window, provide the following information:
    
    Parameter
    
    Description
    
    Name
    
    Give the new framework a name.
    
    Description
    
    (Optional) Describe the new framework.
    
    External link
    
    (Optional) If your custom framework is based on a public standard, you can provide a link to the definition.
    
3.  Click **Save**.
    

### Add a practice to the framework [Copied!](#add-a-practice-to-the-framework "Copy link to this section")

1.  In the landing page, click **New Practice**:
    
2.  In the Practice details window, provide the following information:
    
    Parameter
    
    Description
    
    Title
    
    Give the new practice a name.
    
    Subtitle
    
    Give the new practice a subtitle.
    
    Description
    
    (Optional) Describe the new practice.
    
3.  Click **Add**.
    

### Add a requirement to the practice [Copied!](#add-a-requirement-to-the-practice "Copy link to this section")

1.  In the practice you created, click **Add requirement**.
    
2.  In the Requirement details window, provide the following information:
    
    Parameter
    
    Description
    
    Title
    
    Give the new requirement a name.
    
    Subtitle
    
    Give the new requirement a subtitle.
    
    TL;DR
    
    Provide a short summary of the requirement.
    
    Description
    
    (Optional) Describe the new requirement.
    
3.  Click **Add**.
    

### Add a control to the requirement [Copied!](#add-a-control-to-the-requirement "Copy link to this section")

1.  In the requirement you created, click **Add control**.
    
2.  In the Control details window, provide the following information:
    
    Parameter
    
    Description
    
    Control ID
    
    Give the new control an ID.
    
    Name
    
    Give the new control a name.
    
    Description
    
    (Optional) Describe the new control.
    
3.  Click Add.
