## Overview [Copied!](#overview "Copy link to this section")

By default, Cycode scans for application insights on APIs as of the feature release in April 2025. To get historical data, go to the **Settings** > **Backoffice** > **On-demand scans** page and create a Code Context Analysis scan.

Note

See the list of [languages and web frameworks](/before-you-begin/supported-code-languages) currently supported for API Detection.

## View the APIs [Copied!](#view-the-apis "Copy link to this section")

The API table contains the following information about the tools that were detected in your organization.

Item

Description

Name

Path that includes the name of the API.

Method

HTTP method of the API.

Locations

Source that contains the API, and the endpoints.

Languages

Coding languages where the API was detected.

Frameworks

Coding frameworks where the API was detected.

You can filter the list by repository, language, framework, and search by the API path/name. To see details about a specific API, click the row in the table to open the card.
