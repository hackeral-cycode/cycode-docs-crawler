# Build entities

[](https://gitlab.com/cycodehq/documentation/cycode_docs/-/blob/main/src/content/inventory/build-entities/_index.md "Edit this page")

Updated on Apr 1, 2025 • Published on Oct 21, 2024

* * *

Build entities are the specific components that are involved with the build process of an application. Managing build entities effectively is crucial for ensuring consistent, repeatable, and secure software builds, helping teams maintain code quality and compliance. You can see information for the following build entities in the Inventory view:

*   [Build workflows](/inventory/build-entities/build-workflows)
    
*   [Build reports](/inventory/build-entities/build-reports)
    
*   [Attestations](/inventory/build-entities/attestations)
