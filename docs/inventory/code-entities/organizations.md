From the **Organizations** area of the **Inventory** page, you can view all the organizations in your environment.

![](/assets/code-entities-image-t45ez9ck.png)

## Filters [Copied!](#filters "Copy link to this section")

You can filter the results according to the following options:

*   Source: According to the source code hosting sites with which each organization is affiliated
    
*   Label:According to the labels assigned to the organization
    
*   Cycode Project: According to the Cycode Project
    
*   Plan: According to the type of plan the organization has with the hosting site
    

## Available Details [Copied!](#available-details "Copy link to this section")

The following information is provided in the Organizations table:

Column name

Description

Organization

Contains the name of each organization in your environment

Labels

Labels assigned to the organization (e.g., private)

Source

Contains the source code hosting sites with which each organization is affiliated

Users

The users that belong to this organization

Plan

The type of plan the organization has with the hosting site

Total Violations

The total number of violations in the organization
