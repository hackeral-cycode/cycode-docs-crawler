# Users

From the **Users** area of the **Inventory** page, you can view all of the users with access to the repositories in your environment.

## Filters [Copied!](#filters "Copy link to this section")

You can filter the results according to the following options:

*   Source: According to the source code hosting sites with which each user is affiliated
    
*   Organization: According to the organizations in your environment with which each user is affiliated
    
*   Label: According to the labels associated with the users
    
*   SCM Role: According to the user SCM roles
    
*   Affiliation: According to the user affiliation
    
*   Is Bot: According to whether the user is a bot or not
    
*   2FA: According to whether users have 2FA authentication or not
    

## Available Details [Copied!](#available-details "Copy link to this section")

The following information is provided in the Users table:

**Column name**

**Description**

Source

Contains the source code hosting sites with which each user is affiliated

Username

Contains the username of each user with access to your environment

Full Name

Contains the full name of each user with access to your environment

Organizations

Contains the organizations in your environment with which each user is affiliated

Affiliation

Contains the connection of each user to organizations in your environment (the default filter is Member and Outside collaborator)

Platform Role

Contains the role of each user for organizations in your environment

2FA

Contains whether multi-factor identification is enabled for each user

Total Validations

Contains the total number of validations for each user

Is Bot

Indicates whether the user is a bot
