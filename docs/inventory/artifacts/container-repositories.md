## Filters [Copied!](#filters "Copy link to this section")

Container repositories can be filtered by the following

**Filter**

**Description**

Source

According to the source code hosting sites

Discovery Source

According to the security tool through which the container registry was discovered

## Available Details [Copied!](#available-details "Copy link to this section")

The following information is provided in the Container Registry table:

**Column name**

**Description**

Container Repository

Name of the container repository

Location

The SCM where the container repository is located

Images

The number and names of the container images of the container repository

Tags

The tags associated with the container repository

Related Code Repositories

Code repositories related to this container repository

Discovery Source

The security tool through which the container registry was discovered (e.g., Cycode, Orca, Kubernetes, etc.)
