# Which policies are used for detecting code leaks and secrets?

[](https://gitlab.com/cycodehq/documentation/cycode_docs/-/blob/main/src/content/policies/cycode-policies-faqs/which-policies-are-used-for-detecting-code-leaks-and-secrets.md "Edit this page")

* * *

All the policies listed in the Policies page are used for public code leaks scanning. This means that when a repository is suspected of being a leak, we’ll scan it for secrets.

## Which policies are used to check for password leakage or secrets in the code? [Copied!](#which-policies-are-used-to-check-for-password-leakage-or-secrets-in-the-code "Copy link to this section")

In the Violations page, select Secrets in the Violations Category. This displays all the secret policies in the Cycode platform. Note that all the policies listed are used in code scanning for leaks as well.
