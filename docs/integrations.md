## Introduction [Copied!](#introduction "Copy link to this section")

Cycode provides a wide range of pre-built integrations that are designed for all your DevOps tools, to enable complete supply chain security. Integrations are essentially a unified system composed of different “units” that bring different capabilities and options. Our integrations deploy quickly and easily to deliver immediate value and allow maximum agility across all of the tools that make up your SDLC.

In this section, you’ll find information about the different types of integrations that can be integrated with Cycode, which permissions you’ll need for these integrations, any prerequisites, and how to set them up.

All integrations are set up using the Cycode platform Settings function.
