# Step 4: Connect external security tools to Cycode

[](https://gitlab.com/cycodehq/documentation/cycode_docs/-/blob/main/src/content/cycode-onboarding/getting-started-with-onboarding/step-4-connect-external-security-tools-to-cycode.md "Edit this page")

Updated on May 26, 2024 • Published on May 23, 2024

* * *

If you purchased the Cycode Posture Management module, you can use it to connect all your security tools for visualization, prioritization, and remediation through a single pane of glass, providing correlation, deduplication, data enrichment, and automated workflows.

If you’re not sure whether you purchased this module, check out the [Organization Plan page](/inventory/code-entities/organizations) to see what you purchased.

To learn more about how to connect your external security tools, see [Security tools](/integrations/security-tools) .
