## About the Cycode developer experience [Copied!](#about-the-cycode-developer-experience "Copy link to this section")

Cycode provides a robust developer experience that’s designed by developers for developers. Our goal is to enable you, the developer, to write **secure code fast**. To that end, Cycode integrates with the relevant engineering tools and processes so that you can integrate security into your native environments with no context switching. This allows you to focus on building the features that matter without sacrificing secure coding best practices. It’s a win-win for everyone.

The Developer Experience section is designed for two types of users. Click the user that’s relevant for you to get started:

*   [Security Administrators](/developer-experience/security-administrators/)
    
*   [Developers](/developer-experience/developers/)
