# Add company custom guidelines to PR scans

[](https://gitlab.com/cycodehq/documentation/cycode_docs/-/blob/main/src/content/developer-experience/developers/pull-request-pr-scanning/add-company-custom-guidelines-to-pr-scans.md "Edit this page")

Updated on May 5, 2024 • Published on Apr 15, 2024

* * *

Cycode’s PR scanning feature enables you to add company custom guidelines to your PR scans (for GitHub, GitLab and Azure DevOps).

To use this capability, you’ll first need to add company custom guidelines to either [a Cycode policy or to a custom policy](/policies/company-custom-remediation-guidelines) you create.

The company custom remediation guidelines will then be displayed as follows:

![](/assets/fd0e83c-image.png)

Click the arrows next to Description and Company’s Remediation Guidelines to view their full content.  
![](/assets/efcbdc1-image.png)
