# Pull Request (PR) scanning

[](https://gitlab.com/cycodehq/documentation/cycode_docs/-/blob/main/src/content/developer-experience/developers/pull-request-pr-scanning/_index.md "Edit this page")

Updated on Aug 4, 2024 • Published on Mar 26, 2024

* * *

In the increasingly shift left mode that’s become a prerequisite for successful application security, developers have become key players, and one of the most popular tools available to developers as part of this new configuration is the pull request (PR) scan. Scanning for violations in PRs makes it possible to detect violations early in the development lifecycle.

In this section you’ll find all the information you need about Cycode’s PR scanners and their capabilities.
