# VS Code Extension installation

[](https://gitlab.com/cycodehq/documentation/cycode_docs/-/blob/main/src/content/developer-experience/developers/cycode-ide-plugins/cycode-vs-code-extension-and-compatibles-editors/vs-code-extension-installation.md "Edit this page")

Updated on Nov 14, 2024 • Published on Nov 14, 2024

* * *

1.  To install VS Code, go to VSCode and locate the “Extensions” tab, then search for Cycode in the Extensions: Marketplace.
2.  Open the Cycode vscode extension and click **Install**.  
    ![](/assets/2084a99-image.png)
3.  Click **Next** and follow the steps below.

Once you complete the installation, to scan your projects, you’ll need to [authenticate with Cycode](/developer-experience/developers/cycode-ide-plugins/cycode-vs-code-extension-and-compatibles-editors/authenticating-with-cycode) .
