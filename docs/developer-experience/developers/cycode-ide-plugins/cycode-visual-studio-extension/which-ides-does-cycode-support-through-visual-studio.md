# Which IDEs does Cycode support through Visual Studio?

[](https://gitlab.com/cycodehq/documentation/cycode_docs/-/blob/main/src/content/developer-experience/developers/cycode-ide-plugins/cycode-visual-studio-extension/which-ides-does-cycode-support-through-visual-studio.md "Edit this page")

Updated on Jul 10, 2024 • Published on Jul 3, 2024

* * *

Cycode supports the following IDEs through Visual Studio:

*   Visual Studio 2015
*   Visual Studio 2017
*   Visual Studio 2019
*   Visual Studio 2022 (both AMD and ARM architectures)
