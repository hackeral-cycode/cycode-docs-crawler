## Overview [Copied!](#overview "Copy link to this section")

In Cycode, inventory refers to digital assets or entities within your organization that need to be protected from cybersecurity threats. These can include sensitive data, intellectual property, source code, configurations, and other critical resources that are crucial to your business operations. Cycode specializes in protecting these entities by providing solutions for securing source code repositories and other digital resources from insider threats, unauthorized access, and data breaches.

Note

Only Owners and Admins can access the Inventory view.

The Inventory view displays the entities that contain the different assets in your inventory, so you can view and manage all of the entities in your environment. When you open the Inventory view, you can access the following inventory pages:
