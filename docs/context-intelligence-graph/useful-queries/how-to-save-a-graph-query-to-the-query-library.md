## Overview [Copied!](#overview "Copy link to this section")

This explains how to save a graph query you created to the library for future use.

## Prerequisites [Copied!](#prerequisites "Copy link to this section")

*   User in Cycode has the Owner or Admin role
    
*   Access to the Graph view
    

## Instructions [Copied!](#instructions "Copy link to this section")

1.  In the Context Intelligence Graph, create a query based on the details provided in [Build queries with the query builder](/context-intelligence-graph/build-queries-with-the-query-builder) .
    
2.  After you’re satisfied with your query and the results you have displayed, click the **Save icon -> New Query** option in the top-right of the Context Intelligence Graph.
    
3.  Give your query a name, (optionally) a description, and (optionally) a label, and click **Save Query**.
    

## Outcome [Copied!](#outcome "Copy link to this section")

By following these steps, you have created a Context Intelligence Graph query and saved it for quick access in the future.
