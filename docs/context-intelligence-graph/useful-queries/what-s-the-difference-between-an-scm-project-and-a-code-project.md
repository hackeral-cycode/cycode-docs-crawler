# What's the difference between an SCM project and a code project?

[](https://gitlab.com/cycodehq/documentation/cycode_docs/-/blob/main/src/content/context-intelligence-graph/useful-queries/what-s-the-difference-between-an-scm-project-and-a-code-project.md "Edit this page")

Updated on Jun 24, 2024 • Published on Jun 24, 2024

* * *

A **code project** is a project inside a repository, identified by package.json or any other manifest file.

An **SCM project** is something between organization and a repository, which means it contains users and repositories. Note that an SCM project only works in certain SCMs (such as Bitbucket and Azure).
