## Overview [Copied!](#overview "Copy link to this section")

You can view queries in the graph library, which is where you add your custom queries to Cycode’s prebuilt queries.

## Prebuilt queries [Copied!](#prebuilt-queries "Copy link to this section")

The library contains an extensive list of queries that have been provided by Cycode and are relevant for different capabilities of the Cycode platform, such as SCA, compliance, and secret detection. After you select a query from the library, you can see the query results, modify the query, and add filters or additional queries. You can then save the customized query as a new query to the library and make it available for future use.

## Search for a query [Copied!](#search-for-a-query "Copy link to this section")

In the library, you can search for queries in a number of different ways:

1.  Select **All Queries**, **Custom Queries**, or **My Queries**, and then filter the results (optionally) by selecting tags in the top pane.

![](/assets/c215fe1-image.png)

2.  You can also choose to view all the queries related to **Threat Intelligence**.
    
3.  The Categories function lets you search for queries according to the categories defined in the Cycode platform (see more below): SAST, Cloud Security, CI/CD Security, IaC, Leaks, PCA, NIST SSDF, Build Security, SCA, ISO 27001, Compliance, SOC 2, Code Security, SDLC Inventory, Container Security, Tampering, and Secrets Detection.
    

![](/assets/115e0a1-image.png)

### Predefined queries [Copied!](#predefined-queries "Copy link to this section")

Hovering over each predefined query displays the name of the query and a tooltip, as well as the following information:

*   Description
    
*   Category
    
*   Created by Cycode
