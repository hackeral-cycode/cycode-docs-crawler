## Available Details [Copied!](#available-details "Copy link to this section")

The following information is provided in the Build Workflow table:

**Column name**

**Description**

Build workflow

The name of the workflow

Location

The organization where the workflow is located

URL

The URL of the workflow

Deployed Images

The number and name of images deployed in the workflow

Cimon Installed

An indication of whether Cycode’s Cimon is installed in the workflow

Prevent Mode

An indication of whether prevent mode is enabled in the workflow

Last Run

The date and time the last time the workflow ran
