# Teams

From the **Teams** area of the **Inventory** page, you can view all of the teams with access to the repositories in your environment.

![](/assets/code-entities-image-yq33v8zy.png)

## Filters [Copied!](#filters "Copy link to this section")

You can filter the results according to the following options:

*   Source: According to the source code hosting sites with which each team is affiliated
    
*   Organization: According to the team organization
    
*   Label: According to the labels assigned to the teams
    

## Available Details [Copied!](#available-details "Copy link to this section")

The following information is provided in the Teams table:

**Column name**

**Description**

Team

Contains the name of each team working in your environment

Labels

Contains the labels assigned to the team

Location

Contains the location of the organization of the team

Members

Contains the members of the team working in your environment

Total Violations

Contains the total number of violations associated with the team
