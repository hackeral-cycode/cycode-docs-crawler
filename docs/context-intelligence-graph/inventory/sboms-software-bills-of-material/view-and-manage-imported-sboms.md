## Overview [Copied!](#overview "Copy link to this section")

The Imported SBOMs view is your centralized dashboard for managing all SBOMs imported into Cycode. This view provides a high-level overview of all your third-party software assets, allowing you to track their contents and associated risks in one place.

## View the SBOM list [Copied!](#view-the-sbom-list "Copy link to this section")

The SBOM inventory is listed in a table where each row shows a single imported SBOM. You can see the following information.

Column

Description

Name

Descriptive name assigned to the SBOM during import.

Version

Version of the software asset specified within the SBOM.

Vendor

Third-party vendor or organization that supplied the software.

Business impact

Importance of this asset to your business as assigned during import (High, Medium, Low).

Owner

Internal contact person or team responsible for this asset.

Packages

Total count of unique software components listed in the SBOM.

Last import

Date and time the SBOM was last imported or updated in Cycode.

To hide or show specific columns, click the **Add Column** icon above the table.

The following additional column is hidden by default.

Column

Description

Labels

If an SBOM has associated labels, they are displayed in this column.

## Manage your SBOMs [Copied!](#manage-your-sboms "Copy link to this section")

You can do the following for SBOMs that have been imported into Cycode:

*   Update the current version of the SBOM by uploading a new file
    
*   Create a new version of the SBOM by uploading a new file
    
*   Delete an SBOM from Cycode
