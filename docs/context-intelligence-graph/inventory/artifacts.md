Artifacts are crucial for versioning, traceability, and deployment in software development, as they represent the tangible outputs of the development cycle that can be deployed, tested, and distributed. You can manage the following artifacts using the Inventory view:

*   [Container registries](/context-intelligence-graph/inventory/artifacts/container-registries)
    
*   [Container repositories](/context-intelligence-graph/inventory/artifacts/container-repositories)
