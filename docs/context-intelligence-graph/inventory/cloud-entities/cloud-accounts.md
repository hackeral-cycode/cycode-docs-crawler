Cloud accounts can be filtered by source and by account name.

## Available Details [Copied!](#available-details "Copy link to this section")

The following information is provided in the Cloud Accounts table:

Column

Description

Account Name

Name of the cloud account

Source

The source used for the account

VM Instances

The number and names of the VM instances in the account

Storage Buckets

The number and names of the storage buckets in the account

Kubernetes Clusters

The number and names of the Kuberentes clusters in the account
