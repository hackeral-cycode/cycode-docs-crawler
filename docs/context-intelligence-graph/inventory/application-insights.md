# Application insights

[](https://gitlab.com/cycodehq/documentation/cycode_docs/-/blob/main/src/content/context-intelligence-graph/inventory/application-insights/_index.md "Edit this page")

Updated on Apr 1, 2025 • Published on Dec 2, 2024

* * *

The application insights options in the Inventory view provide a wide range of data to help you safeguard your security posture and ensure that sensitive data doesn’t get exposed.

*   [Technologies](/context-intelligence-graph/inventory/application-insights/technologies) - provides an overview of all the tools being used in your organization, even those that haven’t been approved by your security department. You can see where these tools are being used and the code repositories they contain, among other important information.
    
*   [NHIs](/context-intelligence-graph/inventory/application-insights/non-human-identities-nhis/) - provides a comprehensive overview of all the non-human identities that are used across your organization and whether they contain secrets that may get exposed and compromise your security posture.
