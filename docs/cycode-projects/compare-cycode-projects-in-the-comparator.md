## Overview [Copied!](#overview "Copy link to this section")

The Comparator provides a side-by-side comparison view of Cycode projects so you can evaluate key metrics, such as SLA, MTTR, and open versus closed statuses, alongside other operational visualizations that are only visible at the tenant level in the AppSec dashboards.

Metrics are shown in graph view to expose trends and outliers so you can see potential issues at a glance, and are also provided as a simple count (summary) to show the precise values that you are comparing.

## Add Cycode projects to the view [Copied!](#add-cycode-projects-to-the-view "Copy link to this section")

1.  Click **Add a group or project**.
    
2.  Start typing the name of a project in the search field, or select a project from the list.
    
3.  Click the plus icon to add another project for comparison. You may compare up to five projects at a time.
    

## Comparator widgets [Copied!](#comparator-widgets "Copy link to this section")

You can view the compared metrics in the following widgets. The comparator results can also be exported as a PDF.

Widget

Description

Total open violations

Graph view of the simple count of unresolved violations per project.

Summary of total open violations

Simple count of unresolved violations per project.

Open violations by risk

Graph view of open violations per project according to **Critical**, **High**, **Medium**, **Low**, and **Informational** status

Summary of open violations by risk

Chart of open violations per project, shown according to **Critical**, **High**, **Medium**, **Low**, and **Informational** status.

Open violations by category

Heatmap-style distribution of open violations according to category, for an at-a-glance view of category concentrations per project.

Open violations over time

Graph view of open violations over time, which visualizes the trend per project of unresolved violations.

Summary of open violations over time

Chart of open violations over time per project.

Open violations delta over time

Graph view of the change in the number of of open violations over time, which highlights spikes in the number of violations detected and increased activity in handling violations.

Summary of open violations delta over time

Chart that shows changes in the number of open violations (positive and negative) over time.

SLA status

This graph shows open violations based on SLA status (missed, near to be missed, and on track) so you can monitor compliance with your internal SLAs.

Total open violations

Simple count of open violations per project according to their SLA status.

MTTR

Average time (in days) it takes to fix issues for each project. Lower values indicate faster remediation.

MTTR summary

Table view of the MTTR per project.

Max risk score

Graph view of the highest risk score for an individual violation for each project so you can see the worst-case scenario at a glance.

Max risk score summary

Table view that shows the actual value of the max risk score per project.

## Manage Comparator views [Copied!](#manage-comparator-views "Copy link to this section")

### Export a view [Copied!](#export-a-view "Copy link to this section")

You can export your current Comparator configuration as a PDF to share it with others in your organization. The PDF contains all the widgets in the view, similar to a project dashboard report.

### Save a favorite view [Copied!](#save-a-favorite-view "Copy link to this section")

If you regularly track specific insights, you may want to save your current Comparator configuration, including the selected projects, metrics, and any other defined parameters. You can save a favorite view for personal use in the platform, and you can also share a favorite view with the other members in the same tenant.

1.  At the top right corner of the view, click the **Save view** icon.
    
2.  Configure the following parameters:
    

Parameter

Description

Title

Provide a name for the view.

Side menu color

Select a color for the view for easy identification in the side panel.

Share view with all tenant members

Enable this option if you want the view to be available to other members of your organization.

The saved view appears under the Comparator page in the secondary navigation pane. If the view is public, a globe icon appears next to the name.

### Edit a favorite view [Copied!](#edit-a-favorite-view "Copy link to this section")

*   You can change the name and side color of a saved view.
    
*   If a saved view is private, you can make it public to all members in the tenant. You also copy the link for the saved view (both private and public) to send to another member in the tenant).
    
*   You can delete a saved view.
    

Important

When you make a view public, it can be modified or deleted by other members in your tenant.
