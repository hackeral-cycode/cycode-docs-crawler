# Policy last update indicator

[](https://gitlab.com/cycodehq/documentation/cycode_docs/-/blob/main/src/content/scanners/ast-application-security-testing/sast-static-application-security-testing/sast-policies/policy-last-update-indicator.md "Edit this page")

* * *

Cycode policies, along with any custom policies you create, can occasionally be updated. Such updates may result in different Violation outcomes once the code is rescanned.

To help with this, each policy includes an option to display the “Last Updated” date, using customizable logic.

In the SAST policies list, you can:

*   **Filter policies** by the **Last Updated** date
    
*   View a **visual indicator** for policies updated within the **past 30 days**
    
*   **Hover over the indicator** to see the **exact date** the policy was last updated
    

![](/assets/policy-last-updated.png)

Note

The Last Updated date refers only to changes in the core detection patterns. It does not reflect edits to policy titles or metadata.
