## Overview [Copied!](#overview "Copy link to this section")

This topic describes where to configure and manage the Cycode scanner settings. This ensures that the scan results provide the required information for you to manage your violations.

## Scanner configuration [Copied!](#scanner-configuration "Copy link to this section")

The settings can be accessed by administrators in **Settings** > **Scanners**. Settings are specific to each scanner, and are available under the relevant scanner option in the secondary navigation pane.

For information on how to configure scanner options per scanner, go to:

*   [Configure the secret scanner](/scanners/pipeline-security/secrets/configure-the-secret-scanner/)
    
*   [Configure the CI/CD security scanner](/scanners/pipeline-security/ci-cd-security/configure-the-ci-cd-security-scanner)
    
*   [Configure the SCA scanner](/scanners/ast-application-security-testing/sca-software-composition-analysis/configure-the-sca-scanner/)
    
*   [Configure the SAST scanner](/scanners/ast-application-security-testing/sast-static-application-security-testing/configure-the-sast-scanner/)
    
*   [Configure the IaC scanner](/scanners/ast-application-security-testing/iac-infrastructure-as-code/configure-the-iac-scanner)
    
*   [Configure the container security scanner](/scanners/ast-application-security-testing/container-image-security/configure-the-container-security-scanner)
    

Note

Pull request settings are set per scanner where supported.
